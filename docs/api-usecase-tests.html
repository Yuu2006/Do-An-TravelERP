Loi Mong Muon Khi Test Sai

Cac ma loi HTTP pho bien khi test voi du lieu / quyen sai.

401 Unauthorized

{ "status": 401, "success": false, "error": "UNAUTHORIZED" }

Sai hoac thieu token.

403 Forbidden

{ "status": 403, "success": false, "error": "FORBIDDEN" }

Dung token nhung sai vai tro.

404 Not Found

{ "status": 404, "success": false, "error": "NOT_FOUND" }

Du lieu khong ton tai.
